  Run `npm i` to install the dependencies.

# Ai-women-safety-app
“Guardian Path is an AI‑powered mobile navigation app designed to enhance women’s safety by recommending context‑aware safer routes. It analyzes time, crowd presence, lighting, and risk indicators to provide intelligent route suggestions and real‑time safety alerts.”
